Fase 1 - Olá, Mundo!

1. Qual comando imprime 'Olá, mundo!' em Python?
Resposta: print('Olá, mundo!')
O comando print() exibe texto na saída padrão.
2. Qual tipo de dado representa números inteiros em Python?
Resposta: int
O tipo int representa números inteiros.
3. Qual símbolo usamos para comentários de uma linha em Python?
Resposta: #
Comentários de linha começam com o símbolo #.
4. Como declaramos uma variável em Python?
Resposta: numero = 10
Variáveis são declaradas diretamente sem tipo explícito.
5. Qual comando usamos para receber entrada do usuário?
Resposta: input()
A função input() lê dados do usuário via teclado.

Fase 2 - Variáveis e Tipos

1. Como declaramos uma variável inteira em Python?
Resposta: numero = 10
Não usamos tipo explícito, apenas atribuímos o valor.
2. Qual tipo de dado usamos para armazenar um número com casas decimais?
Resposta: float
O tipo float armazena números decimais.
3. Qual é o tipo da variável x = 'Olá Mundo'?
Resposta: string
Strings representam textos em Python.
4. Como declaramos uma constante em Python (por convenção)?
Resposta: X = 10
Constantes são declaradas com nomes em maiúsculas por convenção.
5. Qual comando usamos para verificar o tipo de uma variável em Python?
Resposta: type(x)
A função type() retorna o tipo do objeto.

Fase 3 - Condicionais e Loops

1. Qual das opções cria um loop que conta de 0 a 4 em Python?
Resposta: for i in range(5):
O loop for itera sobre a sequência de 0 a 4.
2. Qual comando usamos para interromper um loop em Python?
Resposta: break
break encerra imediatamente o loop.
3. Como podemos pular uma iteração específica em um loop?
Resposta: continue
continue pula a iteração atual e segue para a próxima.
4. Qual das opções representa um loop while que executa enquanto x for menor que 10?
Resposta: while x < 10:
Loop while executa enquanto a condição for verdadeira.
5. O que faz o comando 'pass' dentro de um loop?
Resposta: Não faz nada e continua
pass é um comando nulo que não executa ação.
