# 101CODE 💻

Plataforma web gamificada para apoio ao aprendizado de programação, inspirada em recursos de gamificação e trilhas interativas.

O projeto foi desenvolvido como atividade acadêmica do curso de Ciência da Computação da PUC Minas – Poços de Caldas.

---

##  Objetivo

Tornar o aprendizado de programação mais acessível, interativo e motivador, por meio de trilhas de estudo, fases, quizzes, progresso salvo e emissão de certificado.

---

##  Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript
- LocalStorage
- Estruturação de páginas web responsivas

---

## ⚙️ Funcionalidades

- Landing page da plataforma
- Cadastro e login de usuário
- Questionário inicial de perfil
- Dashboard com progresso do usuário
- Trilhas de aprendizado por linguagem
- Fases com quizzes interativos
- Salvamento de progresso com LocalStorage
- Materiais complementares
- Geração de certificado ao concluir fases

---

##  Trilhas disponíveis

- Linguagem C
- Java
- Python

---

## 📁 Estrutura do projeto

- `index.html` — página inicial
- `login.html` — acesso do usuário
- `dashboard.html` — painel principal
- `trilhac.html` — trilha da linguagem C
- `trilhajava.html` — trilha de Java
- `trilhapython.html` — trilha de Python
- `certificado.html` — emissão de certificado
- `css/` — arquivos de estilo
- `js/` — scripts da aplicação
- `img/` — imagens e mascotes

---

## ▶️ Como executar

1. Clone ou baixe este repositório.
2. Abra o arquivo `index.html` no navegador.
3. Navegue pela plataforma e teste as trilhas disponíveis.

Recomendado utilizar o Google Chrome para melhor compatibilidade com o LocalStorage.
