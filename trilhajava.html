Fase 1 - Olá, Mundo!

1. Como imprimimos 'Olá, mundo!' em Java?
Resposta: System.out.println("Olá, mundo!");
Usamos System.out.println() para imprimir texto na saída padrão em Java.
2. Qual tipo armazena números inteiros em Java?
Resposta: int
O tipo int armazena números inteiros em Java.
3. Qual símbolo termina uma linha de código em Java?
Resposta: ;
O ponto e vírgula encerra instruções em Java.
4. Qual estrutura usamos para criar um loop que executa 10 vezes em Java?
Resposta: for (int i=0; i<10; i++) { }
O laço for executa repetidamente o bloco de código 10 vezes.
5. Como declaramos um método em Java?
Resposta: void meuMetodo() { }
Métodos são declarados com tipo de retorno, nome e parênteses.

Fase 2 - Variáveis

1. Como declaramos uma variável do tipo inteiro em Java?
Resposta: int numero = 10;
Declaração padrão de variáveis em Java especificando tipo e valor.
2. Qual a sintaxe correta para um comentário de linha única em Java?
Resposta: // comentário
Comentários de linha iniciam com duas barras.
3. Como imprimimos na tela uma variável em Java?
Resposta: System.out.println(numero);
Impressão de variável usando System.out.println.
4. Como criamos um método que retorna void em Java?
Resposta: void meuMetodo() { }
Método sem retorno declarado com void.
5. Qual palavra-chave usamos para criar uma constante em Java?
Resposta: final
A palavra final indica que o valor não pode ser alterado.

Fase 3 - Condicionais

1. Como escrevemos uma condicional if em Java?
Resposta: if (x > 10) { ... }
Estrutura condicional if utiliza parênteses para a condição e chaves para o bloco.
2. Qual a saída de: System.out.println(3 + 2 + "1"); ?
Resposta: 51
Operação é avaliada da esquerda para direita, somando números e concatenando string.
3. Como declaramos um método em Java?
Resposta: void minhaFuncao()
Declaração de método sem retorno.
4. Qual a forma correta de criar uma classe em Java?
Resposta: class MinhaClasse {}
Declaração básica de classe em Java.
5. Como chamamos o método principal (ponto de entrada) de um programa Java?
Resposta: public static void main(String[] args)
Método main é o ponto de entrada de qualquer programa Java.
