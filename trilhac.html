Fase 1 - Conceitos Básicos

1. Qual das opções imprime 'Olá, mundo!' corretamente em C?
Resposta: printf("Olá, mundo!");
A função printf() é usada para imprimir texto na tela em C. As outras opções são sintaxes de outras linguagens.
2. Qual tipo de dado armazena números inteiros em C?
Resposta: int
O tipo int é usado para armazenar números inteiros, ou seja, sem casas decimais.
3. Qual símbolo termina uma linha de código em C?
Resposta: ;
O ponto e vírgula (;) indica o fim de uma instrução em C, essencial para separar comandos.
4. Qual das opções cria um loop que executa 10 vezes?
Resposta: for (int i=0; i<10; i++) { }
O laço for inicializa uma variável, verifica uma condição e incrementa, repetindo o bloco 10 vezes.
5. Qual comando lê um valor digitado pelo usuário?
Resposta: scanf()
A função scanf() é usada para ler entrada do usuário no console em C.

Fase 2 - Variáveis, Comentários e Bibliotecas

1. Como declaramos uma variável inteira em C?
Resposta: int numero = 10;
Em C, declaramos variáveis especificando o tipo seguido do nome e valor opcional.
2. Como fazemos um comentário de uma linha em C?
Resposta: //comentário
Comentários de uma linha em C começam com duas barras (//).
3. Qual comando usamos para imprimir um número inteiro?
Resposta: printf("%d", numero);
O formato %d é usado em printf para imprimir valores inteiros.
4. Como incluir a biblioteca padrão de entrada e saída?
Resposta: #include <stdio.h>
A biblioteca stdio.h contém funções básicas de entrada e saída em C.
5. Como declaramos uma constante em C?
Resposta: const int x = 5;
A palavra-chave const torna a variável constante, não podendo ser modificada.

Fase 3 - Condicionais e Loops

1. Qual estrutura condicional em C está correta?
Resposta: if (x > 10) { printf("Maior"); }
Estruturas if usam parênteses para condição e chaves para o bloco a executar.
2. Como fazemos um loop infinito em C?
Resposta: for(;;)
Um for sem condição cria um loop infinito.
3. Como declarar um array de 5 inteiros em C?
Resposta: int numeros[5];
Arrays são declarados especificando o tipo, nome e tamanho entre colchetes.
4. Qual operador é usado para comparar igualdade em C?
Resposta: ==
O operador == compara valores para verificar igualdade.
5. Como encerramos um programa com sucesso em C?
Resposta: return 0;
return 0; indica que o programa terminou corretamente.
